# Wing Control Buttons Configuration

**Status:** Coming Soon

This file will contain a pre-configured REAPER snapshot with MIDI mappings for controlling Wing Connector actions from your Behringer Wing custom buttons.

## What Will Be Included

When uploaded, this snap file will provide:

- MIDI mappings for Wing custom buttons → REAPER actions
- Wing Connector actions (Connect, Refresh, Toggle Monitoring)
- Transport controls (Record, Play, Stop, Pause)
- Optimized button layout for efficient workflow

## How to Use (Once Available)

1. Download this snap file
2. In REAPER: `Actions → Show action list → Import/export → Import`
3. Select the downloaded file
4. All mappings will be configured automatically

Until this snap file is available, you can configure manually by following the detailed instructions in [README.md](README.md).
